# Expense Tracker (Python)

## Description

A simple Python-based Expense Tracker that allows users to record and manage daily expenses using CSV file storage.

## Features

* Add expenses with category
* View all expenses
* Running total calculation
* Persistent storage using CSV
* Command-line interface

## Technologies Used

* Python
* CSV File Handling

## How to Run

```bash
python expense_tracker.py
```
